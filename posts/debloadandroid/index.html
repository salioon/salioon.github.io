Debloat android:

adb devices - Muestra todos los Dispositivos Conectados

adb reboot - renicia el movil

adb reboot bootloader- reinicia en el bootloader

adb reboot recovery - reinicia el movil en el modo recovery

adb reboot fastboot - reinicia el movil en modo fastboot

adb shell - entra a la raiz del telefono en una shell

adb shell pm list packages - Muestra la lista de paquetes instalados en tu movil

adb install - instala la .apk desde el archivo

adb uninstall com.packagename - desistala paquetes desde la lista de paquetes

adb shell pm uninstall –user 0 appname

fastboot devices - muestra todos los dispositivos conectados

fastboot reboot - reinicia movil - puedes usar el comando bootloader, recovery, and fastboot

fastboot oem device-info - muestra informacion del estado del bootloader (unlocked or locked)

fastboot oem unlock - este comando permite desbloquear el bootloader

fastboot flashing unlock - este comando desbloquea el bootloader en rom custom por ejemplo pixel y otros moviles

fastboot flashing unlock_critical - esto debloquea el bootloader y sistema particiones - nota: esto no es necesario

fastboot format:ext4 userdata - formatear datos de la particion - note: esto borrara todo el dispositivo

fastboot boot recovery.img - permite bootear en un recovery rapido desde twrp o cualquier cosa

fastboot flash recovery recovery.img - permite flashear un recovery en el movil

fastboot flash boot boot.img - permite flashear sistema boot

fastboot -w - borrra el movil



***si dan errores las particiones o escritura: *error* no queda espacio o no escribe ni hace nada: Solucion ***



fastboot delete-logical-partition product or

fastboot delete-logical-partition product_a

fastboot delete-logical-partition product_b

fastboot --set-active=a



fastboot --set-active=b
