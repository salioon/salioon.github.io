Prerequisitos

Antes de que empiezes el tutorial nesecitas:

Instalr Hugo (ultima version)
Instalar Git

Explicacion de comandos:

Crea esctructura de tu proyecto.

hugo new site quickstart

Cambiar el directorio del proyecto.

cd quickstart

inizilizar git

git init

Clona Ananke theme into en el directorio themes, añade esto a tu Git submodule.

git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke themes/ananke

cambia la linea de configuracion del tema.

echo "theme = 'ananke'" >> config.toml

Ejecutar servidor hugo para ver web.

hugo server

Press Ctrl + C para parar el servidor hugo. añadir contenido

para añadir un nuevo post a la web.

hugo new posts/prueba.md

Hugo creo el archivo en : content/posts

esto es una prueba en la pagina web.

presentacion:

Visit the web website!

ejecutar servidor hugo con los draft.

hugo server –buildDrafts hugo server -D

edita la configuracion del archivo (config.toml)

baseURL = 'http://miweb.github.io/'
languageCode = 'en-us'
title = 'Mi web'
theme = 'ananke'

para comprobar el sitio al completo en el server: hugo server -D

para publicar el sitio ejecuta:

hugo

luego vas la carpeta public copias el contenido en el servidor web personalizado que tengas preparado
